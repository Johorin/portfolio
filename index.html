Introduction

名前：北條　佑弥　生年月日：1999年4月12日生まれ

栃木県にて生まれる。大学に入学するまで両親や祖母の願いもあり地元で生活していたが、都会での生活への憧れがあるため、将来的に一度は首都近辺に住んでみたいと本人は思っている。 高校は栃木県立宇都宮北高等学校を卒業後、国立大学法人宇都宮大学工学部情報工学科に在学中。
いつしか適当に過ごして遊んでばかりの生活をしていたが、大学3年の4月、youtubeにてあるエンジニアのチャンネルの動画をみて、WEB業界やプログラミングの仕事に興味を抱き始め、 それを機にプログラミングの独学を始める。
なんとなく興味が湧いて購入した参考書等は最後までやり遂げた試しが今までなかったが、目標ができてからは参考書も最後までやり遂げられるようになったし、取り組んでいた学習が終わってからも発展的な内容やアウトプットも根気強く長く続けられるようになった。
目標ができれば自分はこんなに出来ることを知った。

以下では私が勉強してきたスキルについて客観的にレベルを判断して頂けるように具体的に説明していく。

Skills

HTML & CSS

HTMLとCSSは大学2年の頃に何故か興味を持っていてなんとなくやろうと思っていたので 初心者向けの参考書を購入し半分位までやってみたがこれといった目標もなかったので 長く続かずにやらなくなってしまった。その後、大学3年になって目標ができてからは、 ProgateのHTML&CSSコースを全コース、さらに道場コースまでやり遂げる。自分で一 から作成したサイトや模写サイトは全て本サイトの"Works"ページに記載する。また、 普段からエディタでのコーディングはEmmet記法やSassを使うようにしており、余裕 がある時にはBEM記法を使って書くこと挑戦もしている。

JavaScript(ES6)

HTMLとCSSに加えてアニメーションを追加できるプログラミング言語とのことでまず ProgateのJavaScriptコースを全コース取り組んだが、直接サイトを作るのに役立つ学 習ではなく、ただ単に文法を学んだだけであった。そこで、JavaScriptの参考書 （確かな力が身につくJavaScript「超」入門） を購入し、実際に自分でサイトを動かしながら学ぶことができた。この参考書によって、 JavaScriptによる基本的なDOM操作を学ぶことができた。しかし、アウトプットがまだ 全くやれていないので今後の学習プランとしては、JS-ProでJavaScriptの知識を網羅 的に身に付けた後にドットインストールでWebアプリの作成に取り組んでアウトプットを こなしていきたいと考えている。

jQuery

先述した参考書にはjQueryにも触れられており、jQueryによるAjax通信を用いた、 Web API（OpenWeatherMap）を利用し、取得した天気情報を基にそのデータを 加工するやり方まで学ぶことができた。とはいってもこの参考書だけで扱っている jQueryの範囲は決して広いものではないので、今後の学習プランとしてはせっかく 月額支払っているProgateでの学習にも取り組み、先述したJavaScriptとセット でアウトプットしていきたいと考えている。

PHP

大学3年になって目標ができてから、初めて自分で率先して取り組んだプログラミング言 語であり、PHPの参考書（スラスラわかるPHP） を購入して学習を進めた。SQLを用いた学習になってからターミナルでの操作が難しく、 うまく動作せず先に進めない事態に陥ってしまった。長時間悩んだが解決しなかったので とりあえずそこでPHPは後にして他のプログラミング言語の学習をすることにした。HTML、 CSS、JavaScriptをざっくり学び終えた後、ProgateでPHPを道場コース含め全コースに 取り組んだ。今後は自分にとっての壁であるSQLをProgateで学びつつPHPのアウトプット にも取り組んでいこうと考えている。

WordPress

ある程度サイトの模写ができるようになってきたので、WordPressも参考書 （いちばんやさしい WordPress 入門教室） を購入し、サーバーのレンタル、独自ドメインを取得するところからレンタ ルサーバーにWordPressを埋め込んで学習を開始した。この参考書では、 サイト運営に関する細やかな情報が記載されており、私が以前知らなかった webサイトにおけるセキュリティに対する理解を深めることができた。私は 以前から、検索してもなかなかヒットしないような難しい大学の授業で困って いる大学生に向けて、学習の助けになるブログ型のサイトを作りたいと思っていた。 というのも、大学の授業で分からない部分を調べようとして何度検索をかけても 「わかりやすい」と心から思えたものに巡り合えたのはyoutuberのヨビ○リ さんの動画くらいのものだったからである。現在、作成中の当ブログを"Works"に 記載しておくので是非そちらもご覧ください。
